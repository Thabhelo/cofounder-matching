# Co-Founder Matching Platform

Built for TechStars for Cofounders - A centralized platform that helps founders find co-founders with complementary skills, discover entrepreneurial resources, and navigate the startup ecosystem.

## Project Status

**Planning Phase** - See [IMPLEMENTATION_PLAN.md](./IMPLEMENTATION_PLAN.md) for detailed implementation plan.

## Overview

This platform helps founders:
- Find co-founders and early team members with complementary skills
- Discover entrepreneurial resources, organizations, and opportunities
- Connect with the startup ecosystem
- Access programs, funding, and mentorship opportunities

## Tech Stack

- **Frontend**: Next.js, TypeScript, Tailwind CSS
- **Backend**: FastAPI (Python)
- **Database**: PostgreSQL with pgvector
- **Auth**: Clerk
- **Deployment**: Vercel (frontend), Render/Fly.io (backend)
